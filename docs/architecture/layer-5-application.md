# Layer 5: Application & API

The application layer is not specified by OCP itself. It represents domain-specific AI applications, products, and agents that build on the lower four layers. Domain-specific extensions (such as ocp-ext:healthcare:dicom-share:v1 or ocp-ext:finance:risk-signal:v1) can be registered in the OCP Extension Registry and layered on top of the core protocol.
